# raw.vives — Resumen para portfolio

## Versión corta

raw.vives es el archivo visual bilingüe de Alex Vicente: una experiencia editorial construida con Next.js que combina narrativa, exploración fotográfica y detalle individual. Diseñé y desarrollé el sistema completo —identidad, motion, archivo, SEO, accesibilidad, pipeline de imágenes y preparación para Cloudflare— con una arquitectura estática y progresiva.

## Mi contribución

- Dirección visual y sistema editorial.
- Arquitectura frontend y componentes React.
- Archivo con filtros, búsqueda y rutas compartibles.
- Motion GSAP, cursor contextual y WebGL progresivo.
- Pipeline WebP/AVIF y política de publicación fotográfica.
- i18n ES/EN, SEO técnico, accesibilidad y sistema de release.

## Resultado

73 rutas estáticas, 60 detalles localizados, 67 tests, artefacto de 39,1 MiB sin fuentes fotográficas y documentación operativa completa. La release está validada localmente y pendiente de despliegue manual autorizado.

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS 4 · GSAP · Three.js · Vitest · Sharp · Cloudflare Workers
