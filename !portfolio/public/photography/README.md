# Imagen de Fotografía

Esta carpeta contiene la imagen principal para tu sección de fotografía.

## Cómo añadir tu imagen:

1. **Guarda tu imagen** en esta carpeta (`public/photography/`) con el nombre: `hero.jpg`

2. **Formatos recomendados**: 
   - `hero.jpg` (recomendado)
   - `hero.png` 
   - `hero.webp`

3. **Tamaños recomendados**: 
   - Relación de aspecto: 16:9 o 21:9 (panorámica)
   - Ancho mínimo: 1200px
   - Optimizada para web (tamaño < 2MB)

4. **Tipo de imagen sugerida**:
   - Una de tus mejores fotografías
   - Que represente tu estilo fotográfico
   - Preferiblemente horizontal/panorámica
   - Con buena composición para texto superpuesto

## Después de añadir la imagen:

La imagen se mostrará automáticamente en lugar del placeholder gradient. Si no añades ninguna imagen, se mantendrá el placeholder con el ícono de cámara.

## Para cambiar la imagen:

Simplemente reemplaza el archivo `hero.jpg` en esta carpeta con tu nueva imagen (manteniendo el mismo nombre).
